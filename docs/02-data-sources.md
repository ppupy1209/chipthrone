# 데이터 소스 설계

chipthrone은 시간대에 따라 서로 다른 소스에서 시세를 가져온다. 핵심은 "실제가"와 "추정가"를 명확히 구분해 표시하는 것.

## 1. 정규장 / NXT — 실제가

### 한국투자증권 KIS Developers (무료)
- REST API, OAuth 토큰 발급 후 사용. 모의/실전 도메인 구분.
- 국내주식 현재가 시세 조회 엔드포인트로 005930(삼성전자), 000660(SK하이닉스) 조회.
- 호출 제한이 있으므로 **스케줄러 단일 폴링 + 캐시**로 보호한다.

### 거래 시간 (KST)
- 정규장: 09:00 ~ 15:30
- NXT 애프터마켓: 15:40 ~ 20:00
- 공휴일/주말은 휴장 → 추정 모드로 전환.

## 2. 야간 / 주말 — 추정가

### Hyperliquid HIP-3 perp (공개 info API, 무료)
- 삼성전자·SK하이닉스·KOSPI200을 추종하는 24시간 무기한 선물(perp)이 상장되어 있다.
- 티커 형식 예: `xyz:SKHX` (정확한 심볼은 구현 시 info API로 확인).
- USD 표시 가격을 **USD/KRW 환율**로 원화 환산하여 추정가로 사용한다.
- 공식 거래소 시세가 아니므로 UI에 항상 "추정" 라벨을 붙인다.

### 환율 소스
- 무료 환율 API(예: exchangerate.host 등) 또는 KIS 환율. 구현 시 확정.

## 3. 시장 상태 판별 로직

```
now(KST) 기준:
  주말/공휴일                → ESTIMATE (Hyperliquid)
  평일 09:00~15:30          → REGULAR (KIS)
  평일 15:40~20:00          → NXT (KIS)
  그 외 평일 시간           → ESTIMATE (Hyperliquid)
```

공휴일 판별은 한국천문연구원 특일 정보 API 또는 정적 공휴일 테이블 사용.

## 4. 폴링 주기 (초안)

| 모드 | 주기 | 비고 |
|---|---|---|
| REGULAR / NXT | 2~5초 | KIS 쿼터 고려 |
| ESTIMATE | 3~10초 | perp 변동성 고려 |

## 5. 검증 필요 항목 (구현 전 확인)

- [ ] KIS 현재가 API가 실시간/지연 어느 쪽인지, 무료 쿼터 정확한 수치
- [ ] Hyperliquid 삼성전자/하이닉스 perp 정확한 심볼명
- [ ] 상장주식수 데이터 소스 (KRX / 공시) 및 갱신 주기
- [ ] 환율 API 무료 한도

> ⚠️ 본 서비스의 추정 시세는 투자 참고용이며, 공식 거래소 가격이 아니다.
